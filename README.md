## Getting Started
### Server Programming
#### Set-Up
- Install [NodeJS](https://nodejs.org/en/)
- Install [MongoDB](https://docs.mongodb.com/getting-started/shell/)
- Install dependency

```
  $ cd ServerProgramming/myapp
  $ npm install
```

#### Get Sensors data and insert to DB
```
  $ node dataGetter.js
```

#### Run Web Server
```
  $ npm start
```
